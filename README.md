# Analise-salario

Neste projeto, embarcaremos em uma jornada completa de análise e previsão de salários nas áreas relacionadas a dados. Para isso, coletaremos e exploraremos dados de diversas fontes, abrangendo pesquisas salariais, sites de empregos e perfis de mídias sociais. Utilizaremos técnicas de Análise Exploratória de Dados (EDA) para revelar padrões e tendências salariais ocultas. Ao longo dessa jornada, nossa caixa de ferramentas incluirá uma variedade de gráficos e visualizações, tornando os resultados da análise facilmente compreensíveis.

À medida que avançamos, empregaremos modelos de previsão de salários que nos permitirão estimar com precisão os salários futuros nas áreas relacionadas a dados. Este projeto visa fornecer insights valiosos para profissionais e empregadores, orientando decisões e estratégias relacionadas a remuneração nesse campo em constante evolução
